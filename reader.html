ReaderÂ¶

Creating a new reader for your experimental facility is easy. To do so, create a new Class that follows the propocol defined by pyropy.ExperimentReader. In other words, create a class that requires as input the filename and folder, and it creates the objects time, temperature, rho and dRho. The way you create those is up to you, but they have to be created at instantiation (this is in the __init__ or in the __post_init__).

You can check the class pyropy.reader_netzsch.ExperimentReaderNetzsch for an example.
